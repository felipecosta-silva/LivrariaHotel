# Projeto LivrariaHotel

## Descrição
Desafio lançado no preparatório Java da Fuctura para fazer um crud de um Hotel.

## Atributos

    - CNPJ
    - Nome
    - Rua
    - Bairro
    - Cidade
    - CEP
    - Capacidade
## Funções

    - Criar novo Hotel
    - Listar todos
    - Busca por id epecífico
    - Atualizar
    - Deletar
## Tecnologias utilizadas

    - Java
    - JPA
    - Spring Boot
    - Lombok
    - MySQL
